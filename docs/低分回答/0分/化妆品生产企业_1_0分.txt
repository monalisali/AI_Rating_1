### 二、企业所得税视同销售调整

**法规依据：**

根据《企业所得税法实施条例》第二十五条<sup>[1]</sup>，企业将货物用于**广告**等用途的，应当**视同销售货物**。

根据国税函[2008]828号第二条<sup>[2]</sup>，企业将资产用于**市场推广或销售**，因资产所有权属已发生改变，应按规定视同销售确定收入。

根据国家税务总局公告[2016]80号第二条<sup>[3]</sup>，企业发生国税函[2008]828号第二条规定情形的，除另有规定外，应按照被移送资产的**公允价值**确定销售收入。

**视同销售调整计算：**

- **视同销售收入**：按公允价值确定。题目给出"市场零售价10万"，此为含增值税价格。题目假设不考虑增值税，因此视同销售收入 = 10万元（假设10万为不含税价，或题目已排除增值税因素）
- **视同销售成本**：会计已将成本8万计入广告费，税法视同销售时也应确认成本8万
- **视同销售纳税调增收入** = 10万（公允价值）
- **视同销售纳税调增成本** = 8万（产品成本）

在A105010《视同销售和房地产开发企业特定业务纳税调整明细表》中：
- 第3行"用于市场推广或销售视同销售收入"：税收金额10万，账载金额0，调增10万
- 第13行"用于市场推广或销售视同销售成本"：税收金额8万，账载金额0，调减8万

### 三、视同销售差额的二次调整（关键步骤）

根据skill方法论中的"调整事项的完整链条"原则，视同销售确认收入10万、成本8万后，产生**差额2万元**。会计仅按成本8万入账广告费，而税法认为该广告宣传行为的经济实质是：企业以公允价值10万的货物换取了10万的广告服务，因此广告费应为10万而非8万。

**二次调整逻辑：**

会计入账广告费 = 8万（按产品成本）
税法认可的广告费 = 10万（按产品公允价值，即视同销售收入）

差额 = 10万 - 8万 = 2万，应**调增广告费2万**。

这一调整在A105000《纳税调整项目明细表》中，通过视同销售差额的二次调整实现——视同销售收入调增10万，视同销售成本调减8万，净效果为调增应纳税所得额2万。但更准确的理解是：视同销售表中收入调增10万、成本调减8万，两者合计净调增2万，而广告费本身无需再单独调增（因为差额已通过视同销售的净调整体现）。

**实务口径**：在纳税申报表填报中，A105010视同销售表的净调整效果（收入调增10万-成本调减8万=净调增2万）已经涵盖了广告费差额的调整。因此，不需要在A105000中再对广告费做额外调增2万。

### 四、消费税视同销售问题

**法规依据：**

根据《消费税暂行条例》第四条<sup>[4]</sup>，纳税人自产自用的应税消费品，用于其他方面的，于移送使用时纳税。

根据《消费税暂行条例实施细则》第六条<sup>[5]</sup>，"用于其他方面"包括用于**广告**等。

根据财税[2016]103号<sup>[6]</sup>，自2016年10月1日起，"化妆品"税目更名为"高档化妆品"，税率调整为**15%**。征收范围包括高档美容、修饰类化妆品、高档护肤类化妆品和成套化妆品。高档美容、修饰类化妆品是指生产（进口）环节销售（完税）价格（不含增值税）在**10元/毫升（克）或15元/片（张）及以上**的。

**情形分析：**

- **情形一**：该化妆品属于**高档化妆品**（生产环节销售价格不含增值税≥10元/毫升或15元/片），需缴纳消费税
  - 根据《消费税暂行条例》第七条，自产自用无同类销售价格的，按组成计税价格计算：
  - 组成计税价格 = (成本+利润) ÷ (1-比例税率) = (8万+利润) ÷ (1-15%)
  - 需根据国家税务总局规定的成本利润率确定利润部分
  - 如有同类销售价格，则按同类销售价格计算：消费税 = 10万 × 15% = 1.5万（假设10万为不含增值税的同类销售价格）

- **情形二**：该化妆品不属于高档化妆品（普通美容修饰类化妆品），自2016年10月1日起已**取消消费税**，无需缴纳消费税

**消费税对企业所得税的影响：**

根据《企业所得税法实施条例》第三十一条<sup>[1]</sup>，税金是指企业发生的除企业所得税和允许抵扣的增值税以外的各项税金及其附加。因此，视同销售缴纳的消费税属于"税金"，可在企业所得税前扣除。

- 若需缴纳消费税1.5万：该消费税应计入"税金及附加"，可在企业所得税前扣除
- 会计是否已入账该消费税？题目未提及，若会计未入账，需调增扣除（即调减应纳税所得额1.5万）

### 五、广告费扣除限额问题

**法规依据：**

根据《企业所得税法实施条例》第四十四条<sup>[1]</sup>，企业发生的符合条件的广告费和业务宣传费支出，不超过当年销售（营业）收入15%的部分，准予扣除。

但根据财政部、税务总局公告[2020]43号<sup>[7]</sup>（适用于2021年1月1日至2025年12月31日），**化妆品制造或销售**企业发生的广告费和业务宣传费支出，不超过当年销售（营业）收入**30%**的部分，准予扣除。

**注意**：财政部、税务总局公告[2025]16号<sup>[8]</sup>自2026年1月1日起执行，2025年度仍适用公告[2020]43号<sup>[7]</sup>的30%限额。

**广告费扣除计算：**

- 本企业为化妆品生产企业，适用30%扣除限额
- 税法认可的广告费 = 10万（视同销售公允价值）
- 扣除限额 = 当年销售（营业）收入 × 30%
- 假设当年销售（营业）收入为X万元，限额 = X × 30%
- 若10万 ≤ X × 30%，广告费全额扣除，无需因限额再做调整
- 若10万 > X × 30%，超出部分需调增，结转以后年度扣除

|------|----------|----------|----------|----------|
| 1 | 视同销售收入（用于广告） | 10万 | - | 实施条例第25条、828号第2条、80号第2条 |
| 2 | 视同销售成本（用于广告） | - | 8万 | 828号第2条、80号第2条 |

| 4 | 消费税（若属高档化妆品） | - | 1.5万（若会计未入账） | 消费税条例第4条、第7条、财税[2016]103号 |

**净调整效果（不考虑限额和消费税未入账）：**
- 视同销售净调增 = 10万 - 8万 = **2万**
- 若高档化妆品消费税1.5万未在会计入账，则额外调减1.5万
- 广告费限额调整视当年收入情况而定

## 核心发现

1. **视同销售调整**：自产化妆品用于广告宣传，企业所得税需视同销售，按公允价值（10万）确认收入，按成本（8万）确认成本，净调增应纳税所得额**2万元**<sup>[1][2][3]</sup>。

2. **消费税视同销售**：需判断该化妆品是否属于"高档化妆品"（生产环节不含税售价≥10元/毫升或15元/片）。若属于高档化妆品，需按15%税率缴纳消费税；若不属于，自2016年10月1日起普通化妆品已取消消费税<sup>[4][5][6]</sup>。

3. **消费税扣除影响**：若需缴纳消费税，该消费税作为"税金"可在企业所得税前扣除<sup>[1]</sup>。若会计未入账该消费税，需调减应纳税所得额。

4. **广告费限额**：化妆品生产企业适用**30%**扣除限额（2025年度仍适用公告[2020]43号<sup>[7]</sup>），而非一般企业的15%。视同销售公允价值10万作为广告费基数，需与当年销售（营业）收入×30%比较，超出部分调增并结转以后年度。

5. **调整完整链条**：视同销售收入调增10万与成本调减8万的净效果（调增2万），实质上就是广告费从成本价8万调整为公允价值10万的差额2万，无需再对广告费做单独二次调增。

---

## 风险评估

1. **高档化妆品定性风险**：题目中"市场零售价10万"是否属于高档化妆品需根据单位价格标准判断（≥10元/毫升或15元/片）。若定性错误，消费税处理将完全不同。

2. **公允价值确定风险**：题目给出"市场零售价10万"，但零售价通常含增值税。题目假设不考虑增值税，实务中需确认该价格是否为不含税价。

3. **消费税会计入账风险**：若企业缴纳了消费税但未在会计上入账"税金及附加"，会导致企业所得税汇算清缴时遗漏扣除。

4. **广告费限额风险**：视同销售收入10万是否计入"销售（营业）收入"基数来计算限额，实务中存在争议。根据申报表填报口径，视同销售收入应计入销售（营业）收入基数。

5. **政策衔接风险**：2025年度适用公告[2020]43号（30%限额），2026年度起适用公告[2025]16号<sup>[8]</sup>（仍为30%限额），限额比例未变但需注意政策衔接。
